# WinCleaner

Простой PowerShell-скрипт `win11_optimizer.ps1` для базовой оптимизации Windows 11. Сейчас это заготовка: внутри функции `Optimize-Windows11` можно разместить нужные команды.

## Требования
- Windows 11
- PowerShell 5+ (встроен в систему)
- Запуск от имени администратора

## Как запустить
1. Откройте PowerShell **от имени администратора**.
2. Разрешите выполнение скриптов в текущей сессии:
   ```powershell
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned
   ```
3. В каталоге репозитория запустите:
   ```powershell
   .\win11_optimizer.ps1
   ```

## Что делает скрипт
- Печатает сообщение «Optimizing Windows 11…».
- Предусматривает блок для ваших оптимизационных команд внутри `Optimize-Windows11`.

## Рекомендации по безопасности
- Перед изменениями создайте точку восстановления или резервную копию.
- Тестируйте на отдельном профиле или виртуальной машине.
