# Win11 Optimizer

Утилита для оптимизации Windows 11 с графическим интерфейсом на PowerShell (Windows Forms).

## Что делает инструмент

Win11 Optimizer предоставляет удобный GUI для выполнения распространённых задач по оптимизации и повышению конфиденциальности Windows 11:

| Вкладка | Возможности |
|---|---|
| **Оптимизация** | Отключение служб SysMain, Windows Search, DiagTrack |
| **Очистка** | Удаление временных файлов (`%TEMP%`), Prefetch, журналов событий |
| **Конфиденциальность** | Отключение рекламного ID, персонализированных предложений, диагностики, отслеживания запуска приложений и советов Windows |
| **Восстановление** | Отмена всех изменений реестра и служб из резервной копии |

## Как запустить

1. Откройте **PowerShell от имени Администратора**.
2. При необходимости разрешите выполнение скриптов:
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
3. Запустите скрипт одним из способов:
   - Дважды кликните `run_win11_optimizer.cmd` (запустит PowerShell с `-ExecutionPolicy Bypass`; при необходимости запросит повышение прав).
   - Либо выполните напрямую:
     ```powershell
     .\win11_optimizer.ps1
     ```

> **Требуется** Windows 11 и права Администратора (`#Requires -RunAsAdministrator`).

## Где хранятся логи и резервная копия

| Файл | Путь |
|---|---|
| Журнал | `%ProgramData%\Win11Optimizer\log.txt` |
| Резервная копия | `%ProgramData%\Win11Optimizer\backup.json` |

Все действия записываются в журнал с временными метками. Резервная копия создаётся автоматически перед каждым изменением реестра или службы.

## Что покрывает Undo (восстановление)

Функция **«Отменить все изменения»** на вкладке «Восстановление» восстанавливает:

- ✅ **Значения реестра** — возвращает исходные значения (или удаляет ключи, которых не существовало).
- ✅ **Службы** — восстанавливает тип запуска (`Automatic` / `Manual` / `Disabled`) и при необходимости запускает службу заново.

**НЕ восстанавливается:**

- ❌ Удалённые файлы (очистка временных файлов, Prefetch, журналы событий).
- ❌ Деинсталлированные приложения.

## Уровни риска

В каждом диалоге подтверждения указан уровень риска:

| Уровень | Описание |
|---|---|
| **НИЗКИЙ** | Изменение легко отменяется; практически не влияет на работу системы |
| **СРЕДНИЙ** | Может затронуть часть функций Windows; рекомендуется создать точку восстановления |
| **ВЫСОКИЙ** | Существенное изменение; обязательно создайте точку восстановления |

## Предупреждение и отказ от ответственности

> ⚠️ **Используйте этот инструмент на свой страх и риск.**
>
> Перед применением любых изменений **рекомендуется создать точку восстановления системы**:
> ```powershell
> Checkpoint-Computer -Description "До Win11 Optimizer" -RestorePointType "MODIFY_SETTINGS"
> ```
